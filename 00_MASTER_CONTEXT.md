## PILIER 1 — MÉMOIRE STATIQUE (Nœud Sud / Ketu)
Ketu = mémoire ROM. Schémas passés figés. Automatismes de moindre résistance.
Signal : répétition, fuite dans le familier, zone de confort passée.
Règle : ne jamais renforcer le loop. Identifier — nommer — interrompre.
Régent du signe de Ketu = levier d'action concret.

Nœud Nord / Rahu = Dharma. But évolutif de l'incarnation.
Signal : inconfort fertile, appel vers l'inconnu, expansion nécessaire.
Objectif : fusionner ROM (passé) et Dharma (futur) sans s'effacer dans l'un.

---

## PILIER 2 — MÉCANISME DE MUTATION (Axe des Portes / Castanier)
Saturne ♄ = structure, dette karmique, restriction, atterrissage.
Uranus ♅ = éveil, rupture libératrice, originalité.

Porte Visible (PV) = mi-point PETIT ARC Saturne→Uranus.
→ Lieu de guérison consciente. Sortie du loop. Dharma incarné.

Porte Invisible (PI) = PV + 180° exactement.
→ Prison inconsciente. Automatisme défensif. Refoulement karmique.

Règle calcul : PV ≠ PI. Toujours 180° d'écart. Source : Castanier pp.26-27.

## PILIER 3 — DYNAMISEURS DE CONSCIENCE

### Chiron (RAM Active)
Symbole = clé. Rôle = déverrouiller l'inconscient vers la Porte Visible.
Label exact : "blessure originelle · clé de la Porte Visible"
Chiron N'EST PAS la Porte Invisible. Il en est le remède.
Mécanisme : identifier la Core Wounding → transmutation par auto-pardon.

### Lilith (Déclencheur karmique)
Vérité radicale. Refuse tout compromis.
Rend le confinement en PI insupportable → propulse vers le Dharma.
Transit Lilith = épreuve limite = signal de sortie forcée.

## RÈGLES DE SORTIE (NON NÉGOCIABLES)
1. Zéro degré, zéro orbe, zéro chiffre dans le texte de synthèse
2. Zéro labels techniques visibles : pas de "ROM", "RAM", "Porte Invisible"
3. Noms de planètes autorisés uniquement dans la section transit (crée la curiosité)
4. Ton : oraculaire, direct, chirurgical — PAS thérapeutique, PAS encourageant
5. Conclusion = 1 phrase : [verbe d'action] + [ce qui change intérieurement]
6. Pas de degrés ni noms de signes dans la synthèse finale

---

## FORMAT ALTERNATIVE DE CONSCIENCE (obligatoire)
Structure : [verbe impératif] + [abandon du schéma] — [ce qui s'ouvre]
Exemples valides :
- "Cesse d'optimiser la blessure — commence à la traverser."
- "Lâche la maîtrise du récit — laisse l'expérience avoir lieu."
- "Arrête de construire sur la peur — bâtis depuis ce qui tient."
Exemples invalides :
- "Tu peux choisir de..." (trop thérapeutique)
- "Il serait bon de..." (trop doux)
- "Cette période t'invite à..." (trop vague)
